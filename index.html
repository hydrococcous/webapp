WebApp vs. App*

Was ist der Unterschied?

App*

Eine App ist eine native Software, entwickelt für ein bestimmtes Betriebssystem.

WebApp*

Eine WebApp ist eine Anwendung die in einem Browser läuft.

Nachteile*

  • eingeschränketer Hardwarezugriff
  • keine Push-Nachrichten
  •  

Vorteile*

  • plattformunabhängig
  • freie Entwicklungsumgebung
  • schnelle Updates
  • nur ein Entwicklungsstrang
  • keine Abhängikeit von Stores
